Объекты в JavaScript

  • Объект в JavaScript - это тип, предназначенный для хранения коллекции объектов разных типов
  • представляет собой одну сущность, которую можно описать набором свойств
  • информация о свойствах хранится в формате ключ: значение
  • сам объект обозначается фигурными скобками: { height: 186, width: 15, gender: male, name: 'John Smith' };

Создание объекта

Существует 2 способа создать объект в JavaScript:

  • let obj = new Object(); - с помощью конструктора, встроенного в JavaScript;
  • let obj = {}; - используя фигурные скобки при создании переменной
