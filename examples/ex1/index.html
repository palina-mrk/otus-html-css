Структура web-страницы

Код web-страницы написан на HTML.

HTML - язык разметки гипертекста, состоящий из тегов (теги - специальные команды, которые обозначают тот или иной элемент страницы). HTML не является языком программирования, т.к. код на HTML не выполняет никаких действий, а только указывает браузеру, как следует отображать каждый из элементов страницы.

<тег атрибут="значение" >

Тег состоит из

  • имени тега
  • атрибутов тега

Атрибут состоит из

  • имени атрибута
  • значения атрибута

Как выглядит программа на JavaScript

Любая программа на JavaScript называется "скрипт". "Sdript" переводится с английского как "сценарий".

  • скрипт на языке JavaScript - это сценарий, выполняющий определённую задачу.
  • скрипт - это сценарий, по которому программа взаимодействует с пользователем. Скрипт реагирует на действия пользователя.

Чтобы выполнить скрипт, необходимо включить его в код web-страницы. Это мождо сделать двумя способами:

  • Включить код JavaScript в html-код web-страницы. Для этого нужно записать его внутри парного тега <script> </script>.
  • Подключить файл с кодом JavaScript к веб-странице. Для этого нужно указать путь к файлу как значение атрибута src парного тега <script> </script>
